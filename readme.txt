== Description ==

MultiD is an easy install plugin that provides multi-language capability to your wordpress theme. Multilingual support is everytime headache for any developer or site designer. Beside design and other technical issues, there are also a lot of SEO concerns which effects your websites visibility on search engines. MultiD will eliminate most of these problems for you. You may immediately start to fill in your content. Installation process will be just a few minutes. As soon as you activated MultiD plugin your whole site is ready for visitors.

MultiD is a young project targeting to provide a simple solution for language problems. Please share bugs and your ideas. We are using and testing MultiD in our projects. It is open source. We hope that it will help you and community for better websites serving multilingual content. We like to do more... If MultiD gain you some time, and make your life easier, please support us with a little donation at our home page.

* MultiD will allow you to translate post, term and all custom types. By managing translated slugs your site will serve unique content.
* You can use it with any theme. Beside your custom dictionary MultiD will manage wordpress locale tags for you.
* MultiD widgets will help you to create a better content.
* SEO asistance settings will solve content related problems.
* MultiD does not require any deep programming skills.

== Installation ==

Intalling MultiD is easy.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings->MultiD Settings screen to configure the plugin.
